# layoffs-data-cleaning
Cleaned and standardized a company layoffs dataset by removing duplicates, handling missing values, standardizing text fields, and formatting dates for analysis.
This repository contains SQL scripts for cleaning and preparing a global company layoffs dataset. The work includes:

Removing duplicate records

Standardizing company, industry, and country names

Handling null or blank values

Converting date strings to proper date format

Removing unnecessary columns for a clean, analysis-ready dataset

These scripts can be reused as a template for data cleaning and preprocessing tasks in SQL.
